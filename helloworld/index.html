Hello World!

04 April 2015

I got the idea to start a blog on New Year's Day. Well, it took me four months to get around to it, but here we go...

I plan on using this blog to serve as a journal and a device to keep myself motivated. My major goal for 2015 is to learn. I often found myself wishing I knew more about something over the past year. Whether that thing was iOS Development, music, art, or something else entirely, it always came down to me "not having the time to learn it".

The Game Plan

Each month, I'm going to pick a topic, and spend an hour each day learning/improving on it. This month, I'm going to learn how to use Sketch, so I can become a better UX designer. I've wanted to try my hand in design for a while now, and I'm stoked to get started.

For the coming months, I came up with a list of things I'd like to learn or improve on

  • Music Theory
  • Arabic
  • Web Development
  • Cooking
  • Ukulele
  • Piano
  • Functional Programming
  • Make an iPhone game

I anticipate the hardest part of this plan will be making the time to do these things. It's super easy to procrastinate and tell yourself you're too tired to learn, but that doesn't get you anywhere. To start, I'm going to start waking up an hour earlier and encorporating learning sessions into my morning routine. Discipline is going to be key here!

See you on the other side,
-Zach
